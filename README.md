# 🌤️ Python Weather App

A simple **Python command-line application** that fetches the current weather information for any city using the [OpenWeatherMap API](https://openweathermap.org/api).

---

## Features
- View current temperature (°C)
- See humidity percentage
- Get weather conditions (clear sky, rain, clouds, etc.)
- Works for cities worldwide
- Easy command-line interface
